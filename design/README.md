# design

Living technical design and contracts for chit. The implementation cites
several of these directly. They are kept current; when a doc and the code
disagree, the code is the source of truth.

These are not the user-facing docs. The published documentation lives in
`apps/site/content/docs/` and on the website.

- `manifest-schema.md` - the chit manifest contract.
- `mcp-surface.md` - the stepwise MCP surface.
- `audit-log.md` - the audited-run record.
- `studio.md` - the Studio editor.
- `convergence-log.md` - the loop / convergence log.
- `studio-nodes.md` - Studio node state markers.
